# OhmMap.com | Global Energy & EV Infrastructure Platform

OhmMap is a high-end visualization platform designed for the next generation of smart cities and electric vehicle (EV) charging networks.

## 🚀 Vision
To bridge the gap between energy consumption data and geographic infrastructure, providing real-time insights into grid resistance and power distribution.

## 🛠 Tech Stack
- **Framework:** Next.js 2026
- **Map Engine:** Mapbox GL (3D Grid Visualization)
- **Styling:** Tailwind CSS (Ultra-Dark Tech Theme)
- **Animations:** Framer Motion
- **Icons:** Lucide React

## ⚡ Key Features
- **Global Consumption Map:** Real-time power load simulation for major cities (Dubai, London, NYC).
- **EV Charging Network:** Interactive markers with occupancy status, pricing, and charger types.
- **Dynamic Analytics:** Visual data representation for renewable energy percentage and grid stability.
